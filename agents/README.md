# Agents Directory

This directory contains the implementation of all AI agents in the recruiting system.

## Structure

- `supervisor_agent.py` - Orchestrates the multi-agent workflow
- `sourcing_agent.py` - Finds and ranks candidates
- `screening_agent.py` - Evaluates resumes and qualifications
- `outreach_agent.py` - Generates personalized outreach messages
- `scheduling_agent.py` - Optimizes interview scheduling
- `evaluation_agent.py` - Synthesizes feedback and recommendations
