# Results Components

This directory contains components for visualizing final results and recommendations.

## Components

- **ResultsVisualization.tsx** - Main results page orchestrator
- **TopCandidatesRanking.tsx** - Top 3 candidates showcase
- **SkillMatchChart.tsx** - Radar chart for skill matching
- **OutreachPreview.tsx** - Generated outreach messages
- **InterviewSchedule.tsx** - Interview scheduling display
- **FinalRecommendations.tsx** - AI-generated recommendations
