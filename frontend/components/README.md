# Components

This directory contains React components organized by feature:

## Structure

- `ui/` - shadcn/ui base components (button, card, input, etc.)
- `layout/` - Layout components (Header, Footer, Navigation)
- `job/` - Job posting related components
- `workflow/` - Workflow dashboard components
- `candidates/` - Candidate pipeline components
- `results/` - Results visualization components

Components will be created incrementally in Phase 7.2-7.4
