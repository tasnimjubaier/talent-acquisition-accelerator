# Candidate Components

This directory contains components for displaying and managing candidate data.

## Components

- **CandidatePipeline.tsx** - Main candidate list with filtering/sorting
- **CandidateCard.tsx** - Individual candidate display
- **CandidateDetailModal.tsx** - Detailed candidate view
- **CandidateFilters.tsx** - Filter controls
