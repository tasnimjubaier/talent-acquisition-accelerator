# Workflow Components

This directory contains components for displaying real-time workflow status and agent activity.

## Components

- **WorkflowDashboard.tsx** - Main dashboard orchestrating all workflow components
- **AgentStatusCard.tsx** - Individual agent status display
- **ProgressTracker.tsx** - Visual progress indicator
- **CostTracker.tsx** - Real-time cost monitoring
- **ActivityFeed.tsx** - Live activity stream
